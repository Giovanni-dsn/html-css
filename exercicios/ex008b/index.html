Outras formatações

Códigos

Nós podemos delimitar uma área para mostrar um código escrito com a tag <code>. Por exemplo:

num = 0 while num <= 6: print(f'{num} ...') num += 1 print('Terminei de contar !')

Só que dessa maneira, o código não ficará formatado, apenas as letras serão impressas no modo monoespaçado para identificar no texto que trata-se de um código.

Caso queira que o código fique na formatação desejada, é necessario usar a tag <pre> e colocar a tag <code> dentro dela. Por exemplo:

        
    num = 0
    while num <= 6:
        print(f'{num} ...')
        num += 1
    print('Terminei de contar !')
