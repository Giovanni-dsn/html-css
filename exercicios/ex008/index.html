Texto Marcado

Você pode marcar um texto usando a tag <mark>

Textos pequenos

Você pode deixar um texto pequeno com a tag <small>

Texto deletado (Riscado)

Podemos deletar um texto, ou seja, deixá-lo riscado usando a tag <del>. Isso indica que o texto "deletado" deve ser lido, mas desconsiderado pelo leitor

Texto inserido (sublinhado)

Você também pode inserir um texto(sublinhar) usando a tag <ins>. Isso indica que o leitor deve considerar o texto "inserido" um pouco mais.

Texto sobrescrito

Para alocar termos como x20 + 3. Usamos a tag <sup> com o "20" dentro.

Texto subescrito

Para alocar termos como H2O. Usamos a tag <sub> com o "2" dentro.
