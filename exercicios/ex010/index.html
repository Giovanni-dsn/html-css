links externos

Para apontar o link externo num conteúdo html basta coloca-ló dentro da tag <a> e colocar a URL em "href", como feito em repositório do guanabara

links internos

Você pode adicionar um link interno com a mesma tag <a> sem a necessidade de indicar a url inteira, se for dentro do próprio site/servidor/arquivo. Basta indicar o nome do arquivo, por exemplo Página 002
